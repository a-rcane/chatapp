# Chatapp

 Real-time-chat application using Django

## Installation

 * Clone repo
 * Create venv for python (3.11)
 * Install requirements 
   ```bash
      $ pip install -r requirements.txt
   ```
 * Run Migrations
   ```bash
      $ python -m manage makemigrations
      $ python -m manage migrate
   ```
 * Run server 
   ```bash
      $ python -m manage runserver 8000
   ```
